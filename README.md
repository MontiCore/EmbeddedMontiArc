## Usage

To use the different sub-project as maven dependency, add the following to your pom's `<dependencies>`:

```xml
<dependency>
    <groupId>montisim</groupId>
    <artifactId>sub-project-name</artifactId>
    <version>${revision}</version>
</dependency>
```
