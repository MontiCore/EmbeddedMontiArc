## Usage

To use this sub-project as maven dependency, add the following to your pom's `<dependencies>`:

```xml
<dependency>
    <groupId>montisim</groupId>
    <artifactId>vehicle</artifactId>
    <version>${revision}</version>
</dependency>
```
